  references remain. ✓

- QuorumOptions: NOT retired. Live struct definition at

  QuorumRuntime composes into.

The AgentRunRuntime / CompareRuntime / QuorumRuntime types added by
beads 17/24/etc. are an outer-API layer that internally still
constructs the legacy options types (e.g.
cli/internal/agent/compare_adapter.go:60-78 explicitly:
`opts.RunOptions = RunOptions{...}`). Until follow-up beads excise
those internal sites, the analyzer's pattern 2/3 rules cannot
reach silent.

`ci` block and the GitHub Actions workflow with no additional
changes.
