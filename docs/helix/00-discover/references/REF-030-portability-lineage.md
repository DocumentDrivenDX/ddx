# OCI / POSIX / SQL — Portability Lineage

Each of these standards proved that a stable, narrow contract surface beats a feature-complete vendor lock-in. Cited as the historical evidence base for DDx's contract-first stance: the value lives in the protocol (bead JSONL, agent harness contract), and implementations are interchangeable.
