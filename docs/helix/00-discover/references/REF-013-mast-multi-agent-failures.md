# MAST — Multi-Agent System Failure Taxonomy (2025)

Empirical study cataloguing how multi-agent LLM systems fail in practice — most failures are coordination/verification breakdowns, not raw model capability gaps. Justifies DDx's emphasis on explicit handoffs (beads), clear acceptance criteria, and review gates over loosely-coordinated agent swarms.
