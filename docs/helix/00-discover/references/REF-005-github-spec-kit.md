# GitHub Spec Kit

GitHub's reference implementation of Specification-Driven Development (SDD). Provides templates, prompts, and workflow scaffolding for treating PRDs and technical plans as the primary, executable artifacts that generate implementation. Cited as a foundational tool/methodology source for DDx and HELIX framing.
