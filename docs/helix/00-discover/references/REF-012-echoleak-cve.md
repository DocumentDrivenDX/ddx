# EchoLeak (CVE-2025-32711)

Real-world demonstration of indirect prompt injection escalating to zero-click data exfiltration in a production AI assistant. Cited as concrete evidence that LLM agents need bounded actions, output filtering, and trust boundaries — not just careful prompting.
