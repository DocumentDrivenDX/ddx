# AWS Kiro

Kiro is an AWS spec-first AI IDE that materializes a "specs → design → tasks → code" pipeline. Used as evidence that major platform vendors are converging on spec-driven, document-anchored workflows for AI agents — a key piece of the software-factory framing.
