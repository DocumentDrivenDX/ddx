# Self-Refine (Madaan et al., NeurIPS 2023)

Establishes iterative self-critique-and-revise as a robust quality-lifting pattern for LLMs. Underwrites HELIX's review gates and DDx's review-step escalation: generate, critique, revise — preferably with a stronger reviewer than producer.
