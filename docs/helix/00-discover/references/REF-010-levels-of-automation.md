# Sheridan & Verplank — Levels of Automation (1978)

Origin of the 10-level automation taxonomy used across HCI and autonomous systems research. Provides the conceptual frame for HELIX's supervisory dispatch and DDx's bounded-action model: how much agency the agent gets, and where the human stays in the loop.
