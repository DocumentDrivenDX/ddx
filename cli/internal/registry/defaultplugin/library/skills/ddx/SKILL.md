# DDx

DDx (Document-Driven Development eXperience) is a CLI platform for
document-driven development. This is the embedded default-package stub
that ships with the binary so `ddx init` can install the `ddx` plugin
offline through the same registry installer used for remote installs.

The full skill content is synced into this location from the canonical
source tree at build time. See
`docs/helix/02-design/plan-2026-05-13-ddx-skill-package-layout.md` for
the migration sequence.
