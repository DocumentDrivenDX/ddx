## ddx work clear-cooldowns

Bulk-clear queue-drain cooldowns

### Synopsis

clear-cooldowns scans beads with an active queue-drain cooldown and clears
them in one pass. Use after a systemic issue (e.g., a push-layer failure) is
resolved so blocked beads re-enter the execution queue without per-bead --unset loops.

Requires --all or --status to prevent accidental bulk clears.

```
ddx work clear-cooldowns [flags]
```

### Examples

```
  # Clear all active cooldowns
  ddx work clear-cooldowns --all

  # Clear only push_failed cooldowns
  ddx work clear-cooldowns --status push_failed

  # Preview without modifying state
  ddx work clear-cooldowns --all --dry-run
```

### Options

```
      --all             Clear cooldowns on every bead with retry-after set
      --dry-run         Print what would be cleared without modifying state
  -h, --help            help for clear-cooldowns
      --status string   Clear only beads where last-status matches this value
```
